■JSF（Java Server Faces) 
特徴･･･MVCモデルを採用している。Webアプリケーションの開発などで多く利用されるフレームワーク
メリット･･･GUIとの親和性がある。MVCが採用されているので分業ができるので開発効率が上がる。

■Spring Framework
特徴･･･様々な方面や業務環境での開発をカバーしている。世界的にもシェアがたまっている。
メリット･･･DI(Dependency Injection)という仕組みが採用されており、保守性が高い

■Play Framework
特徴･･･RubyのフレームワークであるRuby on RailとPythonの代表的フレーム枠であるDjangoの概念を取り入れたフレームワーク。Javaと親和性の高いScalaという言語電も使用することができる。
メリット･･･軽量で生産性が高い

■Java EE
特徴･･･大規模システム向けのフレームワーク
メリット･･･ServletやJSP、Javaの各種APIなど、企業システムの構築に必要な機能がまとめられている。
